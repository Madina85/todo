Это текстовая страница

  • Открываем статистические файлы в папке main.(стили,медиа)
  • название папки должно быть "static"
  • Что бы потключиться к статистическому файлу в html документе самом верху должны записать {% load static%}
  • link rel="stylesheet" href="{% static 'styles/test.css' %}"
  • В settings.py пишем "STATIC_URL = '/static/' STATIC_ROOT = BASE_DIR / 'static'
